install this dependencies:

-Windows
Make sure you have Python installed. If not, download and install Python from the official website

pip install mediapipe
pip install pyqt5
pip install pickle
pip install numpy
pip install opencv-python
pip install openai






-Linux(ubuntu/Debain)

Install Python via the package manager if not already installed:

sudo apt update
sudo apt install python3 python3-pip


Install essential build tools:

sudo apt install build-essential


pip3 install mediapipe
pip3 install pyqt5
pip3 install pickle
pip3 install numpy
pip3 install opencv-python
pip3 install openai




-macOS:

Install Homebrew if not already installed. Open Terminal and run:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install Python via Homebrew:

brew install python@3.9

Run the following command to install MediaPipe via pip:

pip3 install mediapipe
pip3 install pyqt5
pip3 install pickle
pip3 install numpy
pip3 install opencv-python
pip3 install openai
